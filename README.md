# Ki Pkdas — Responsive E-Commerce Website

A fully responsive, multi-page e-commerce website for a clothing brand, built using pure **HTML**, **CSS**, and **JavaScript**, with live product data fetched from the **FakeStore API**.

---

## 🌐 Live Preview

> Open `index.html` in your browser or run locally with:
> ```bash
> npx serve .
> ```
> Then visit: `http://localhost:3000`

---

## 📁 Project Structure

```
e commerce site/
├── index.html          # Home page
├── shop.html           # Full product catalog
├── cart.html           # Shopping cart
├── blog.html           # Blog posts
├── about.html          # About us + video
├── contact.html        # Contact & map
├── features.html       # Service features
├── banner.html         # Promotional banners
├── style.css           # Global stylesheet
├── script.js           # JavaScript logic & API
└── imagefolder/
    ├── product/        # f1-f8, n1-n8 (product images)
    ├── blog/           # Blog post images
    ├── about/          # About page images + video
    ├── banner/         # Promotional banners
    ├── features/       # Service feature icons
    └── pay/            # Payment logos
```

---

## ✨ Features

- **Premium UI** — Glassmorphism header, CSS variables, smooth transitions
- **Responsive Design** — Fully mobile-friendly with hamburger menu
- **16 Local Products** — Featured (f1–f8) & New Arrivals (n1–n8)
- **Live API Products** — Global Collection fetched from [FakeStore API](https://fakestoreapi.com/)
- **Shopping Cart** — Dynamic cart badge counter on item add
- **Multi-page Navigation** — 8 fully functional pages

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Flexbox, Grid, Animations, CSS Variables
- **JavaScript (ES6+)** — DOM manipulation, Fetch API
- **Font Awesome 6** — Icons
- **FakeStore API** — Live product data

## 📦 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/thakurSaurabh93/E-commerce-site.git
   ```
2. Navigate to the folder:
   ```bash
   cd E-commerce-site
   ```
3. Open `index.html` in your browser, or serve locally:
   ```bash
   npx serve .
   ```
